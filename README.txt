INSTRUCTIONS:
=============
Double click the file "runscript" to run the application; that's all!


USAGE:
======
Keep the application running while you're laddering!
Every ~2 minutes it'll check for any new replays, and automatically upload them http://sc2replaystats.com!




This is all open source!
